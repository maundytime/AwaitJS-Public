# AwaitJS Public

This is a public template repo for AI agents to write AwaitJS widgets.

## Clone

```bash
git clone https://github.com/maundytime/AwaitJS-Public.git
cd AwaitJS-Public
npm install
```

## Use With AI Agent

```text
Read `skill.md` first and follow it strictly.
Then implement or modify `widget/index.tsx` based on my request.
```

`skill.md` is the main instruction file for agents in this repo.
